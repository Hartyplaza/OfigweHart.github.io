Predicting Bankruptcy in Polish Companies

This project explores bankruptcy prediction in Polish companies using the Poland Bankruptcy dataset. The data, provided in ARFF format within a ZIP archive, includes various financial attributes relevant to company performance and financial health. The analysis involves building and comparing two machine learning models: Random Forest and Gradient Boosting Trees. These models were chosen for their ability to handle complex, high-dimensional dataset. The project includes data preprocessing steps, handling class imbalance through oversampling and undersampling. This analysis aims to deliver insights that can aid in identifying companies at risk of bankruptcy, with potential applications in financial risk assessment and early warning systems.
